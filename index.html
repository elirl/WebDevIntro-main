HTML

Bueno, HTML es un lenguaje de marcado para hacer una pagina web, es decir, con este lenguaje tu asignas que es cada cosa en tu pagina web, por ejemplo: con la etiqueta "p" puedes asignar parrafo, problablemente este parrafo que estas leyendo este marcado con una etiqueta "p". Otra cosa a considerar es que la mayoria de las etiquetas se deben abrir y cerrar, pero hay algunas que no necesitan cerrarse. Ejemplo: 

Se cierra: <p> </;p>
  No se cierra: <img>
Otra cosa muy importante en HTML, son los atributos, un atributo son palabras especial que determinan el uso o comportamiento del elemento en cuestion. Te dejo los mas importantes (rellenados como ejemplo):

- href="facebook.com"
-name="Nombre para metadata"
-charset="UTF-8"
-content="width=device-width, initial-scale=1.0"
-src="link de una fotografia"
-id="identificacion para un elemento" (CSS)
-class="clase de un elemento" (CSS)
